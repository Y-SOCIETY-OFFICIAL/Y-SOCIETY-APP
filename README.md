# Y-SOCIETY-APP

**Y-SOCIETY-APP** aik Kivy/Python based Android learning app hai jo specially cyber security students aur beginners ke liye design ki gayi hai.

Ye app simple, clean, aur future-ready concept ke sath banayi ja rahi hai taa ke learners ko security, terminal basics, aur practical lab environment ka idea mil sake.

## Features

- **Termux Learning**
  Termux ke basic commands, setup, aur useful learning steps ko samajhne ke liye.

- **Security Lab**
  Ek practice-style learning area jahan students cyber security concepts ko safely explore kar sakte hain.

- **Android Friendly**
  Ye app Android devices ke liye build ki ja rahi hai using Kivy and Buildozer.

- **Beginner Friendly UI**
  Simple interface jo new students ke liye easy ho.

- **Future Expansion**
  Is project me baad me quizzes, tutorials, tools, notes, aur offline modules add kiye ja sakte hain.

## About the Project

Y-SOCIETY-APP ka goal ye hai ke students ko cyber security ki dunia me step-by-step guide kiya jaye.  
Roman Urdu me:  
Ye app un students ke liye hai jo Termux seekhna chahte hain, security basics samajhna chahte hain, aur ek practical learning environment me grow karna chahte hain.

English me:  
This app is meant to become a simple cyber security learning platform for Android users, especially beginners who want to explore Termux, security basics, and hands-on lab style learning.

## Tech Stack

- Python
- Kivy
- Buildozer
- Android SDK / NDK

## Run Notes

Main file `main.py` ko run karne ke liye Kivy environment chahiye.  
Android APK banane ke liye Buildozer use hota hai.

## Project Goal

Y-SOCIETY-APP ka main aim ek educational cyber security app banana hai jo modern, useful, aur students ke liye helpful ho.
